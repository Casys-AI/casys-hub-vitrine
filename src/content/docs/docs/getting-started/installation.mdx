One command to install. No build step, no peer dependencies.

## Install

<Tabs>
  <TabItem label="Deno (recommended)">
    ```bash
    deno add jsr:@casys/mcp-server
    ```
  </TabItem>
  <TabItem label="npm">
    ```bash
    npx jsr add @casys/mcp-server
    ```
  </TabItem>
</Tabs>

Published on [JSR](https://jsr.io/@casys/mcp-server) (`@casys/mcp-server`). Works with both Deno and Node.js.

## Requirements

| Runtime | Minimum Version |
|---------|----------------|
| **Deno** | 2.0+ |
| **Node.js** | 20+ |
| **TypeScript** | 5.0+ (recommended) |

The package is Deno-first. Node.js support is available via the JSR npm compatibility layer.

## Verify it works

Create a `server.ts` file:

```typescript
import { McpApp } from "@casys/mcp-server";

const server = new McpApp({
  name: "test",
  version: "0.0.1",
});

console.log(`Ready — ${server.getToolCount()} tools registered.`);
```

Run it:

<Tabs>
  <TabItem label="Deno">
    ```bash
    deno run --allow-net server.ts
    ```
  </TabItem>
  <TabItem label="Node.js">
    ```bash
    npx tsx server.ts
    ```
  </TabItem>
</Tabs>

<Aside type="tip">
  If you see "Ready — 0 tools registered.", you're all set. Head to the Quick Start to build something real.
</Aside>
